# Machine Learning Pipeline Assignment 
## Project Description This project builds a beginner machine learning pipeline to predict customer purchase behavior. 
## Dataset The dataset contains customer information such as age, city, income, website visits, previous purchases, membership level, and purchase status. 
## Steps Completed 
1. Loaded and inspected the dataset
2. Cleaned missing values and duplicates
3. Defined features X and target y
4. Split the data into training and testing sets
5.  Applied preprocessing using StandardScaler and OneHotEncoder
6.  6. Trained a baseline Logistic Regression model
7. Generated first predictions ## How to Run
1. Open the notebook in Jupyter or Google Colab.
2. Install required packages if needed.
3. Run all cells from top to bottom.
## Main Result The baseline model successfully completed the full workflow and generated predictions. 
## Responsible AI Reflection This model is only a learning example. It should not be used for real customer decisions without checking data quality, bias, fairness, privacy, and business consequences.
